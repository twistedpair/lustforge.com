Version 226.0.0 introduces a breaking change for Cloud Spanner's `gcloud beta spanner rows` command, now requiring `--flags-file` for array values with the `--data` flag. New features include an App Engine command group for configuration migration and the beta release of the Cloud Firestore Emulator. Additionally, Cloud Endpoints deployment behavior has been adjusted regarding service enablement.

## Breaking Changes

- Updated `--data` flag of `gcloud beta spanner rows` to reject array values; `--flags-file` should now be used to specify array values.

## New Features by Service

### App Engine

- Added command group `gcloud beta app migrate-config` for migrating App Engine configuration files (`datastore-indexes.xml`, `cron.xml`, `queue.xml`, `dispatch.xml`) to their YAML counterparts.

### Cloud Firestore

- Released Cloud Firestore Emulator version 1.2.1, marking its first beta release.

## API Changes

### Cloud Endpoints

- Modified `gcloud endpoints` deployment to no longer automatically enable the Endpoints meta-service or the produced service, unless the service is being created for the first time during deployment.
