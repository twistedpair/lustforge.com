Version 245.0.0 introduces two breaking changes by removing deprecated Kubernetes Engine flags and properties, making `new_scopes_behavior` the default. Key new features include the `gcloud alpha survey` command, new export options for Cloud Asset Inventory, beta support for Cloud SQL flags in Cloud Run, and new `pause`/`resume` commands for Cloud Scheduler jobs. Additionally, the App Engine Java SDK was updated to version 1.9.74, and the Compute Engine `--instance-redistribution-type` flag was promoted to beta.

## Breaking Changes

- Removed deprecated `--enable-cloud-endpoints` flag from Kubernetes Engine.
- Removed `container/new_scopes_behavior` property from Kubernetes Engine (new_scopes_behavior is now default behavior).

## New Features by Service

### Cloud SDK

- Added `gcloud alpha survey` command to help improve the `gcloud` command-line tool experience by answering a customer satisfaction survey.

### Cloud Asset Inventory

- Added `--output-path-prefix` flag to `gcloud beta asset export` command.
  - Flags: `--output-path-prefix`

### Cloud Run

- Cloud SQL flags are now available in beta for `gcloud beta run` commands.

### Cloud Scheduler

- Added `gcloud beta scheduler jobs pause` and `gcloud beta scheduler jobs resume` commands.

### Compute Engine

- Promoted the `--instance-redistribution-type` flag of `gcloud compute instance-groups managed create` and `gcloud compute instance-groups managed update` to beta.
  - Flags: `--instance-redistribution-type`

### Kubernetes Engine

- Modified `gcloud container clusters create` and `gcloud container node-pools create` to allow simultaneously specifying both `--scopes` and `--service-account` flags.
  - Flags: `--scopes`, `--service-account`

## API Changes

### App Engine

- Updated the Java SDK to version 1.9.74.
