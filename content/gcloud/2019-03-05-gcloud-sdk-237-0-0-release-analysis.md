This release primarily focuses on bug fixes, including a critical fix for a Windows installer crash, BigQuery timestamp formatting, and Cloud Dataproc field visibility. It also updates key components like `gsutil` to 4.37 and `kubectl` to 1.11.7 (adding more versions), while introducing new Compute Engine features such as reduced disk size limits, new options for compute commitments, and enhanced `instance-groups` listing.

## New Features by Service

### Compute Engine

- Added the `resources-accelerator` option to `gcloud beta compute commitments create`.
- Added `INSTANCE_TEMPLATE` and `VERSION_NAME` columns to the output of `gcloud compute instance-groups managed list-instances`.

### Kubernetes Engine

- Added extra `kubectl` versions: `1.9`, `1.10`, `1.11`, `1.12`, and `1.13`.

## API Changes

### Compute Engine

- Promoted the `--server-binding` flag of `gcloud compute sole-tenancy node-templates create` to beta.
- Reduced the lower bound for data disk sizes from `10GB` to `1GB` for `gcloud compute instances create`.
- Promoted the `all` option of the `--ports` flag for `gcloud compute forwarding-rules` to GA.
